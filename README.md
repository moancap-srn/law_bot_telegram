Профессиональный Telegram-бот для автоматизации приёма заявок от клиентов юридической компании.

📋 Описание

Бот позволяет клиентам оставлять заявки на консультацию в удобной форме, а администраторам — управлять заявками, получать статистику и выгружать данные.

✨ Возможности

Для клиентов:
- 📝 Простая форма заявки (4 шага)
- 📱 Отправка телефона через кнопку или вручную
- ⏰ Выбор удобного времени для звонка (3 временных интервала)
- ✅ Мгновенное подтверждение отправки
- 🔔 Уведомления о статусе заявки

Для администраторов:
- 📊 Статистика заявок (сегодня/неделя/всего)
- 📥 Выгрузка всех заявок в TXT файл
- ✅ Принятие/отклонение заявок в 1 клик
- 🔔 Уведомления в отдельный чат

🚀 Быстрый старт

Требования
- Python 3.11 или выше
- Telegram Bot Token (от [@BotFather](https://t.me/BotFather))
